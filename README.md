Markdown语法发布博客前，你需要有一个编辑器来写博客，Gridea中内置了文本编辑器，采用的是Markdown标记语言，而非我们平常使用的Word那种富文本格式。相比富文本格式，Markdown有一个最大的优点：让人更关注写作本身，而不是关注排版。使用Word写作时，经常会花费较多时间在排版上，而Markdown足够简单，去除了Word内置的冗余样式，选择少了，你也没必要纠结于使用哪种样式。<img
